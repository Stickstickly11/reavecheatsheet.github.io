CSS Cheatsheet Table

CSS Tag CSS Name Definition
color: Color The color tag defines the color of the text of the website.
background color: Background Color Background Color refers to, well the background color of the particular piece of the website you are working on.
border: Border The border tag refers to the border around the box of what you are working on. It can have many different forms and colors.
height: Height The height tag refers to the height of the box that you are currently working on and will extend the border.
width: Width The Width tag refers to the width of the box you are currently working on and will extend the border.
font-family: Font Type The font-family tag will change what font you are using for that particular piece of the website.
font-weight: Font Boldness The font-weight tag refers to how bold, or not bold, a particular piece of font is.
